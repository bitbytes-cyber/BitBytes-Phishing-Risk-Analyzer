# BitBytes – Intelligent Phishing Risk Analyzer

**Be the Best Version of Yourself**

BitBytes is a Python Flask-based web application that analyzes messages
and URLs for phishing risks using explainable security heuristics.

## Features
- Phishing Risk Score (0–100)
- Risk Level (Low / Medium / High)
- Detailed detection reasons
- Analyst-style explanation
- Actionable security recommendations
- Neon-themed SOC-style UI

## Technologies Used
- Python
- Flask
- HTML/CSS (embedded)
- Rule-based phishing detection

## How to Run
```bash
pip install -r requirements.txt
python app.py
